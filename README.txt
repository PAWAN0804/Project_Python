1: Basic Calculator

Overview:  
This project is a simple calculator developed in Python. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The program demonstrates the use of input handling, pattern matching (match-case), and exception handling to create an interactive console-based application.

Features:
(a): User Input: Prompts the user to enter two numbers and select an operation.
(b): Arithmetic Operations: Supports addition (+), subtraction (-), multiplication (*), and division (/).
(c): Pattern Matching: Uses Python’s match-case structure to handle different operations.
(d): Error Handling: Catches invalid inputs (non-numeric values) and displays appropriate error messages.
(e): Invalid Operation Handling: Provides feedback if the user enters an unsupported operation.

Technology Stack:
Python (Core language features: input(), int(), match-case, try-except)
